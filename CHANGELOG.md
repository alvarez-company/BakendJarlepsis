# Changelog

Todos los cambios notables en este proyecto serán documentados en este archivo.

El formato está basado en [Keep a Changelog](https://keepachangelog.com/es-ES/1.0.0/),
y este proyecto adhiere a [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Template de Pull Request para GitHub
- Workflow de CI/CD con GitHub Actions
- Scripts de verificación de configuración Git (bash y PowerShell)
- Guía de configuración GitFlow (`SETUP_GITFLOW.md`)
- Mejoras en `.gitignore` para mejor seguridad

### Changed
- Actualizado `.gitignore` con mejores prácticas de seguridad
- Agregadas exclusiones para archivos sensibles y generados

## [1.0.0] - 2025-01-XX

### Added
- Sistema de autenticación JWT
- Gestión de usuarios y roles
- Sistema de inventario completo
- Gestión de materiales, bodegas, oficinas y sedes
- Sistema de movimientos de inventario
- Sistema de traslados
- Sistema de chat y notificaciones
- Documentación completa de API con Swagger
- Configuración Docker
- Migraciones de base de datos
- Sistema de logging con Winston
